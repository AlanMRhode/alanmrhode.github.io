"Fat Cat Smuggling"

A Shipping Application

This collaborative assignment was to create a Rails application for shippers to help them organize all of the ships in their ﬂeet with a user login system. The application is able to allow users to sign up, login and add new boats to their proﬁle, which are meant to be managed by that user. Boats must have a unique name and locations and must be pulled from a hard-coded list. Jobs must have a unique name and must be above $1000 for each job. Cargo description must be at least 50 characters and origin and destinations must be pulled from a hard-coded list.
